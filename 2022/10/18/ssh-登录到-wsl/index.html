局域网内主机登录到其他主机的 wsl

wsl 到 windows 的端口映射

在 wsl 中通过 ifconfig 查看 ip，一定要使用该 ip，不要使用 windows 网络设置中 vEthernet switch 或者 vEthernet wsl 的 ip 地址。

假设该 ip 为 192.168.186.222

在 windows 的防火墙高级设置中，在入规则中开放 50022 端口。
